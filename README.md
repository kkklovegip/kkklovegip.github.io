# 微光手记 · Hexo 博客

基于 Hexo 的中文个人博客，使用自定义 `quiet` 主题，通过 GitHub Actions 自动部署到 GitHub Pages。

## 常用命令

```powershell
npm.cmd install
npm.cmd run server
npm.cmd run build
```

文章位于 `source/_posts/`，使用 Markdown 编写。
